# Wiktionary raw data (English dictionary)

- **Source**: [kaikki.org English dictionary](https://kaikki.org/dictionary/English/) — JSONL, one JSON object per line.
- **Download**: https://kaikki.org/dictionary/English/kaikki.org-dictionary-English.jsonl (≈2.7GB). Save as `kaikki.org-dictionary-English.jsonl` in this directory, or pass `--wiktionary-path /path/to/file.jsonl`.
- **For testing**: Use `--wiktionary-line-limit 50000` to read only the first 50k lines.
