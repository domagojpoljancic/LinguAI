# CEFR raw data

- **Source**: [Open Language Profiles CEFR-J](https://github.com/openlanguageprofiles/olp-en-cefrj) — English vocabulary with CEFR levels.
- **File**: `cefrj-vocabulary-profile-1.5.csv` (optional). If missing, the ingestion script can fetch it with `--fetch-cefr`.
- **Columns**: headword, pos, CEFR, CoreInventory 1, CoreInventory 2, Threshold.
